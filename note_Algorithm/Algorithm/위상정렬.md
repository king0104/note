## 위상정렬 : topological sorting

### 개념

- 위상정렬 : 방향 그래프의 모든 노드를 **'방향성에 거스르지 않도록 순서대로 나열하는 것'** 이다.

  #### 사용하는 목적

  : 위상 정렬을 해야하는 떄는, 어떤 작업의 ''순서''를 나타낼 때 유용하다.  



### 용어

- 진입차수(내차수, Indegree) : 특정한 노드로 **들어오는** 간선의 개수
