### 비트연산자

<비트연산자, 그리고 나누기, 나머지>

- 정수 // 16 : 이진수 기준, 하위 네자리 버리기
- 정수 % 16 : 이진수 기준, 하위 네자리만 남기기

- 이진수의 의미
