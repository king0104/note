## 플로이드 워셜

```python
n, m = map(int, input().split())
graph = [list(map(int, input().split())) for _ in range(n)]

# k : 거쳐가는 노드
for k in range(n):
    # i : 출발 노드
    for i in range(n):
        # j : 도착 노드
        for j in range(n):
            if graph[i][k] + graph[k][j] < graph[i][j]:
                graph[i][j] = graph[i][k] + graph[k][j]
```
