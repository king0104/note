- DFS / BFS



- **DFS** ( 입력 : 시작 노드)
  1. 현재 노드 방문
  2. for문으로 전체 노드 탐색
     1. if (전체 노드 중)**map[현재노드] [다음노드] = 1 && visit[다음노드] == false** 이면,
        1.  dfs(다음노드)



- **BFS** (입력 : 시작 노드)

  1. 현재 노드 방문
  2. 큐에 삽입
  3. while(!q.isEmpty())
     1. 큐에서 하나 빼기
     2. if( **map[현재노드] [다음노드] = 1 && visit[다음노드] == false** ) 이면,
        1. 방문
        2. 큐에 삽입

  - *tip* : 방문 = 큐에 삽입
